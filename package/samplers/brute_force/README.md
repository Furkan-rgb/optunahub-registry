## Class or Function Names

- BruteForceSampler

## Example

```python
import optuna
from optuna.samplers import BruteForceSampler


def objective(trial):
    x = trial.suggest_float("x", -5, 5)
    return x**2


sampler = BruteForceSampler()
study = optuna.create_study(sampler=sampler)
study.optimize(objective, n_trials=10)
```

## Class or Function Names

- QMCSampler

## Example

```python
import optuna
from optuna.samplers import QMCSampler


def objective(trial):
    x = trial.suggest_float("x", -5, 5)
    return x**2


sampler = QMCSampler()
study = optuna.create_study(sampler=sampler)
study.optimize(objective, n_trials=10)
```
